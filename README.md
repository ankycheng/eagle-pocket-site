# Eagle Pocket — Landing Site

Landing page, privacy policy, and support page for [Eagle Pocket](https://eagle-pocket.ankycheng.com), an iOS app for browsing Eagle image libraries.

## Structure

```
index.html              Landing page
styles.css              All styles (Liquid Glass theme)
privacypolicy/index.html   Privacy policy
support/index.html         Support & FAQ
assets/                 Icons, screenshots, App Store badge
```

## Hosting

Static HTML/CSS — deployed on Cloudflare Pages.
